# InstaVest 🚀

India's first short-video crowdfunding platform for startups.

---

## Deploy to Vercel (Free — takes 3 minutes)

### Option A: Drag & Drop (Easiest)
1. Go to [vercel.com](https://vercel.com) and sign up with Google
2. Click **"Add New Project"**
3. Click **"Upload"** and drag this entire `instavest` folder
4. Click **Deploy**
5. ✅ Your site is live at `instavest-xxxx.vercel.app`

### Option B: Via GitHub (Recommended for future updates)
1. Create a free account at [github.com](https://github.com)
2. Create a new repository called `instavest`
3. Upload all files in this folder to the repo
4. Go to [vercel.com](https://vercel.com) → **Add New Project** → **Import Git Repository**
5. Select your `instavest` repo → Click **Deploy**
6. ✅ Live! Every future update you push to GitHub auto-deploys.
